# Atividades de Computação Distribuída

Este repositório reúne atividades propostas na disciplina de **Computação Distribuída**
do curso de **Ciência da Computação** da **Universidade Presbiteriana Mackenzie**.

As implementações exploram conceitos fundamentais de **sistemas distribuídos**,
**comunicação entre processos (ICP)** e **diferentes arquiteturas**, com foco no aprendizado prático.
